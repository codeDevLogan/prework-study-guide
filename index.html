HTML

  • HTML Notes:
    • The 'Style' element (Internal CSS): This element, located in the Head element contains the CSS rules for the webpage.
    • The 'head' element contains data pertaining to the site.
    • The 'body' element contains data and elements shown to the user.
    • The 'link' element allows the developer to link to external webpages.
      • The 'rel' attribute specifies the relationship between the current document and the linked document resource.
      • The 'herf' attribute specifies the location (URL) of the external resource. This attribute uses a relative path this means the location is derived from the current location.
    • The 'img' element allows the developer to display an image.
    • The 'h' family of elements (h1-h6) are descending size and section notational elements.
    • The 'ul' element prints unordered lists.
    • The 'li' element pairs to the 'ul' element in that it is the 'list item' li being short hand for it.

CSS

  • CSS Notes
    • Inline CSS: Apply Styling to a specific HTML tag and to that element alone.
    • The Style element (Internal CSS): This element, located in the Head element contains the CSS rules for the webpage.
    • External CSS: This Style Sheet is typically prefered, allowing developers to keep their CSS rules in a seperate file.
    • Selectors: Define the elements or attributes to which the rules, or declaractions, will apply.
      • The * Selector: This selector is a wild card allowing the declatrations made in the declaration block to apply to all elements in the effected webpage.
    • The Box Model: The Box Model
    • Declarations ({}): contain two important components: the CSS property we want to apply and the value of the property.
    • A period preceeding an selector designates that selector as a Class selector. So any HTML element with the class proceeding the period will be effected by the rules found within the selector's declaration box.
    • Assigning Colors in CSS: Assigning colors in CSS can be done many ways: semantic references (gray, blue) and hexidecimally with the color's hexidecimal code.
    • CSS Properties:
      • Color: Defines the color we want to use on the element.
      • Margin: Defines the amount of space we want around the outside of an element.
      • Padding: Defines the amount of space we want inside the element.
      • text-align: Defines the anchor point for the alignment of the text in the affected element.
      • border: Assigns a border outlining the element.
      • box-shadow: Adds a shadow effect around an element's frame. You can set multiple effects seperated by commas. The shadow is described by x and y offsets relative to the element, blur and spread radius, and color.

JavaScript

  • JavaScript Notes
    • Comments: These are snippets of text that can be added along with code. The browser ignore text marked as comments (/* */ = multiline comments) (// = single line comments)
    • If Else Statements: If Else statements check if the statement inside the parentheses is true. If it is true then the if is triggered, if it is false, the else is triggered.
    • Functions: These let you piece meal your code by removing it from the mainline function.
    • Document: document.querySelector();
    • For Loops: For loops are useful functions that repeat tasks a specified number of times.
    • Events:
      • addEventListener()
    • Operators:
      • Addition(+): Can add both numbers as well as combine two strings.
      • Subtraction, Multiplication, and Division(-, *, /): These all do what you'd expect from basic math.
      • Assignment(=): This assigns a value to a variable.
      • Strict Equality(===): This performs a test to see if two values are equal, returning true or false depending on the result.
      • Not, Does-not-equal(!, !==): This returns the logically opposite value of what it preceeds. When used intandem with an equality operator it tests whether or not the two values are not equal.
    • Variables:
      • String: A sequence of text to signify the value as a string, enclosed in single quotationmarks.
      • Number: A number.
      • Boolean: A true/false statement. The words true and false are special keywords that do not need quotes.
      • Array: This is a structure that allows you to store multiple values in a single reference.
      • Object: This can be anything. Everything in JavaScript is an object and can be stored in a variable.

Git

  • Git Notes
    • git status: Displays which branch you are currently on.
    • git checkout -b branch-name: Creates a new branch with the branch name and switches to it.
    • git add -A: commits all files in the working branch.
